# Lève-toi et Code

Landing page SaaS fictive dont l’objectif est de proposer un plan simple pour progresser en programmation en 2 semaines, à travers une approche 100% pratique (exercices + mini-projets).

## Objectif du projet

Ce projet a été réalisé dans le cadre de la séance 3 (projet de synthèse) du module Responsive Design.

Il met en œuvre :

- une conception Mobile First,
- un responsive desktop par progressive enhancement,
- une structure HTML sémantique,
- une attention à la performance perçue et mesurée,
- une justification des choix techniques et UX.

## Fonctionnalités principales

- Hero avec message clair et CTA unique
- Preuves rapides (bénéfices)
- Bloc d’avis utilisateurs
- Programme détaillé (desktop)
- Section inspiration du projet
- Newsletter (desktop)
- Footer légal
